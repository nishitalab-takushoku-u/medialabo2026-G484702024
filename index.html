「Webコンテンツ作成」の成果

本テーマについて

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的です。到達目標は次の4つです。

  1. 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる。
  2. コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できる。
  3. モーション解析・編集ができる。
  4. HTML等を用いたWebページ作成ができる。

なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

各授業回の学習事項

第1回 WWW と HTML

第1回は WWW と HTMLについて学びました。Web ページを提供する仕組みのことをWWWと言い、HTMLはWebページの構造を記述するための書式らしいです。今回、はお試しでWebページの基礎を学び、HTMLでコードを打ってみました。

演習の成果:演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回の授業では、CSSによるさまざまなWebページのデザインについて学びました。CSSを活用することで、文字の装飾やレイアウトの変更、色彩の工夫など、多種多様な表現ができることを知り、とても驚きました。今後はこの授業を通して知識や技術を身に付け、見やすく華やかなデザインのWebページを作れるようになりたいです。

演習の成果:演習:CSSの活用
